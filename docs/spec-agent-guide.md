# Руководство по написанию промптов для Spec Agent

## Как работает Spec Agent

Spec Agent обрабатывает ваш исходный промпт в два этапа:

1. **`Clarify()`** — анализирует промпт и возвращает до 5–7 уточняющих вопросов, если не хватает критичных данных. Каждый вопрос содержит подсказку с вариантами ответа.
2. **`Build()`** — строит структурированную спецификацию (`Spec`) на основе промпта и ваших ответов.

Чем полнее исходный промпт — тем меньше вопросов задаст агент и тем быстрее запустится пайплайн.

---

## Обязательные секции

Эти поля критичны: без них агент гарантированно задаст вопрос или пайплайн остановится.

| Секция | Поле `Spec` | Что указать |
|---|---|---|
| **Субъект / персонажи** | `characters` | Кто или что главный объект. Человек, предмет, существо, абстракция. |
| **Окружение** | `environment` | Где и когда происходит сцена. Место, время суток, погода. |
| **Настроение** | `mood` | Эмоциональный тон. Меланхолия, эпичность, тревога, радость. |
| **Визуальный стиль** | `style` | Кинематографический, аниме, фотореализм, масляная живопись и т.д. |
| **Формат вывода** | `output_format` | `image`, `video` или `sequence`. Определяет выбор провайдера. |
| **Соотношение сторон** | `aspect_ratio` | `16:9` (горизонталь), `9:16` (вертикаль/reels), `1:1` (квадрат). |

---

## Опциональные секции

Добавляйте их, чтобы получить более точный результат и избежать лишних вопросов.

| Секция | Поле `Spec` | Примеры значений |
|---|---|---|
| **Работа камеры** | `camera_angle` | широкий план, крупный план, птичий глаз, уровень земли |
| **Движение камеры** | `camera_motion` | статика, медленный отъезд, панорама, облёт |
| **Интенсивность движения** | `motion_intensity` | `subtle`, `moderate`, `dynamic` |
| **Освещение** | `lighting` | золотой час, жёсткий неон, студийный свет, контровой |
| **Цветовая палитра** | `color_palette` | приглушённые охры, холодные синие, монохром, пастель |
| **Длительность** | `duration_sec` | только для видео, в секундах (3–10) |
| **Ограничения** | `restrictions` | что явно не должно быть в кадре |
| **Референсы** | `references` | описания стиля или URL изображений |

## Дерево уточняющих вопросов

Агент задаёт вопросы по приоритету. Критичные поля блокируют пайплайн — агент спросит их первыми.

```
Промпт получен
    │
    ├── output_format не указан?
    │       → "Что нужно сгенерировать? [image / video / sequence]"
    │
    ├── aspect_ratio не указан?
    │       → "Какое соотношение сторон? [16:9 / 9:16 / 1:1 / другое]"
    │
    ├── style не понятен?
    │       → "Какой визуальный стиль? [кинематографический / аниме / фото / живопись / другой]"
    │
    ├── mood отсутствует?
    │       → "Какое настроение сцены? [эпичное / меланхоличное / тревожное / радостное / нейтральное]"
    │
    ├── environment неоднозначен?
    │       → "Уточните окружение: время суток, место, погода"
    │
    ├── duration_sec не указан (для video)?
    │       → "Длительность видео? [3 / 5 / 8 / 10 секунд]"
    │
    └── camera_angle/motion хочется уточнить?
            → "Как работает камера? [статика / отъезд / панорама / облёт]"
```

Максимум 5–7 вопросов за итерацию. Агент не задаёт вопросы, если поле можно разумно вывести из контекста.

## Чеклист хорошего промпта

Перед отправкой проверьте:

- [ ] Есть субъект — кто или что в кадре
- [ ] Есть окружение — где и когда
- [ ] Указан формат: `image` или `video`
- [ ] Указано соотношение сторон
- [ ] Задан стиль или настроение (хотя бы одно)
- [ ] Нет взаимоисключающих требований (например: "фотореализм в стиле аниме")

---

## Примеры

### Слабый промпт

```
красивый закат
```

Агент задаст все критичные вопросы: что генерировать, стиль, соотношение, настроение, детали окружения. Минимум 4–5 вопросов.

---

### Хороший промпт — изображение

```
одинокий астронавт стоит на красной пустынной планете в сумерках,
вдали — два горизонта с низкими горами, фотореалистичный стиль,
тёплые оранжево-красные тона, широкий план, image, 16:9
```

Агент сразу строит `Spec` без вопросов. Все критичные поля заполнены.

---

### Хороший промпт — видео

```
одинокий астронавт на красной пустынной планете в сумерках,
кинематографический стиль, camera slowly pulls back revealing the horizon,
меланхоличное настроение, тёплая оранжевая палитра, video, 16:9, 8 секунд
```

`output_format` = video, `aspect_ratio` = 16:9, `duration_sec` = 8, `camera_motion` = медленный отъезд — всё в промпте.

---

### Промпт с ограничениями и референсами

```
японский сад с кленами момидзи осенью, рассвет, туман над прудом,
стиль студии Гибли, мягкий боковой свет, пастельные красно-золотые тона,
без людей, без текста, image, 1:1
```

Поле `restrictions` = ["без людей", "без текста"] — агент передаст их в `negative_prompt` при формировании `ExecutionPayload`.

---

## Шаблон промпта

Скопируйте и заполните нужные строки:

```
[субъект/персонажи] [действие или состояние]
[окружение: место, время, погода]
[настроение: эмоциональный тон]
[визуальный стиль]
[освещение] [цветовая палитра]
[работа камеры: угол, движение]
[ограничения: что не должно быть]
[output_format: image | video | sequence], [aspect_ratio: 16:9 | 9:16 | 1:1]
[duration_sec: N секунд]   ← только для video
```
